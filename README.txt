View Team Dashboard PRO v3.3 – Správa obchodníkov
=================================================
Novinky:
- Sekcia „Obchodníci“ (len pre manažéra): pridávanie, úprava, mazanie používateľov.

- Navigácia a login fungujú bez reloadu, menu skryté pred prihlásením.
