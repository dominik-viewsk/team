View Team Dashboard PRO v2.7 BASE
=================================
Táto „BASE“ verzia je čistý, stabilný základ s funkčným:
- prihlásením (manažér/obchodník),
- menu (Dashboard/Reporty/Ciele/Obchodníci),
- prepínaním sekcií a odhlásením.

Ako používať:
1) Rozbaľ ZIP a nahraj všetko do repozitára dominik-viewsk/team (GitHub Pages).
2) Otvor https://dominik-viewsk.github.io/team a stlač Ctrl+Shift+R.
3) Prihlásenie: 
   - Manažér: dominik@viewsk.com / viewadmin2025
   - Obchodníci: lukac@viewsk.com, illesova@viewsk.com / view2025

2) Formulár Reportu (týždenný) + ukladanie do localStorage
3) Ciele (editácia cieľov manažérom)
4) Správa obchodníkov (pridanie/odobratie)
5) Export/Import CSV
