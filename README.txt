View Team Dashboard PRO v2.8-report
====================================
- Menu/hlavička/foother skryté pred prihlásením.
- Po prihlásení sa menu zobrazí podľa role (manažér všetko, obchodník bez "Obchodníci").
- Modul Reporty pre všetkých: týždeň (číslo + rozsah v zátvorke), oslovenia, dohodnuté stretnutia, realizované stretnutia, ponuky, objednávky.
- Automatické uloženie do localStorage.
- Manažérska tabuľka všetkých reportov (zvýraznenie riadkov s >=2 objednávkami).
- Toast notifikácia "✅ Report uložený".
- Dashboard upozorní, ak ešte nemáte report na aktuálny týždeň.
